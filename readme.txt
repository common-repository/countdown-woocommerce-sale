== Description ==

= For the time sale product of Woocommerce, countdown will be displayed. =

It can view the WooCommerce time sale as countdown.
Discount amount, discount rate, date of sale finish can be displayed.

= Tutorial Video =
[youtube https://www.youtube.com/watch?v=f_gn1Ix2LNg]

== Installation ==

1. Upload `countdown-woocommerce-sale` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
